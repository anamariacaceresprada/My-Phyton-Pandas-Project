# 🐍 Python Data Analysis Project – Retail & Sales Insights

This project was completed as part of a **Data Technician Bootcamp**, where I used **Python (Pandas)** to explore, clean, and analyse retail and sales datasets.

The project focuses on transforming raw data into meaningful insights using data manipulation techniques and visualisation tools.

---

## 🚀 Key Skills Demonstrated

### 📊 Data Analysis with Pandas
- Filtering and subsetting data using:
  - `.loc[]` for label-based selection  
  - `.iloc[]` for position-based selection  
- Selecting specific columns and rows for targeted analysis  

---

### 🧹 Data Cleaning & Preparation
- Handling missing data using:
  - `.dropna()` to remove incomplete records  
  - `.fillna()` to replace missing values  
- Preparing datasets for accurate analysis and reporting  

---

### 🔄 Data Manipulation
- Grouping and aggregating data using:
  - `.groupby()` to summarise key metrics  
- Sorting and organising data with:
  - `.sort_values()` for clearer insights  
- Transforming datasets to answer business questions  

---

### 📈 Data Visualisation
- Creating visual insights using:
  - Built-in Pandas plotting  
  - `matplotlib` for charts and graphs  
- Presenting trends and patterns in a clear and accessible way  

---

## 💡 Project Applications

Using these techniques, I was able to:

- Analyse **retail and sales data** to identify trends and patterns  
- Segment data to understand customer or product behaviour  
- Clean and structure datasets to improve data quality  
- Generate insights to support decision-making
